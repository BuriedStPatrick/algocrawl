# Algoritme crawl

## Navigation

* Minimum 100 meter før nyt sted
	* Skal tage første sted 
* Roterende retning
	* Hvert kryds
	* Hvis sted er på anden side af vej, disregard
* Roterende retningshusker
	* Hvis personen ikke kan huske retning, skal de drikke et shot
	* Hvis personen ikke kan huske retning, og de lyver omkring det og en anden kan huske, drikke 2 shots

## Start

* Start ved rådhuspladsen

## Rotation

1. Venstre
2. Ligeud
3. Høre

* Hvis en retning ikke er mulig, skip

## Sted-kriterier

* Ingen entre
* Skal kunne købe alkohol (alkoholbevilling)
* Et sted man kan sidde
* Max ventetid 10 min
* Når den sidste har drukket første genstand, skal vi gå umiddelbart efter
